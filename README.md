# Operational Amplifier Circuits – LTSpice Simulation
This project demonstrates the simulation and analysis of fundamental Operational Amplifier (Op-Amp) circuits using LTSpice. The objective is to understand the behavior of different amplifier configurations and verify their output waveforms through simulation.

# Circuits Implemented

Inverting Amplifier

Non-Inverting Amplifier

Summing Amplifier

Difference Amplifier

Integrator Amplifier

Differentiator Amplifier

# Tools Used

LTSpice

# Project Overview

Each circuit was designed and simulated in LTSpice with appropriate input signals and supply voltages. The output waveforms were analyzed to verify the theoretical behavior of the operational amplifier configurations.

Learning Outcomes

Understanding of analog circuit design

Practical experience with LTSpice simulation

Analysis of Op-Amp waveform responses
